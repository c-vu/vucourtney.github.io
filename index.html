Projects

MIPS Assembler/Linker and CPU

C program that parses MIPS pseudo-instructions and translates them into TAL assembly code and hexadecimal machine instructions. The linker (written in MIPS) turns machine instructions into an executable file by relocating symbols.

Created a virtual central processing unit that takes hexadecimal machine instructions and performs arithmetic, register writes, and memory writes to fully functionally execute MIPS instructions. Pipelined the CPU to increase performance.


Java program that alphabetically sorts words in input files given a customizable alphabet. Uses the trie data structure to do so.

Using this alphabet, it uses ternary search tries, priority queues, and data on word frequency to find top matches for user-inputted word prefixes (essentially giving autocomplete suggestions).
